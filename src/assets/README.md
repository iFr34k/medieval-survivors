# Game assets

- **Character assets** (sprites, animations) live in a folder per character, e.g. `Knight/`, `Wizard/`.
- **Character-specific docs** (PixelLab IDs, art notes, download instructions): keep a `PIXELLAB_<Name>.md` (or similar) **inside that character’s folder**.
  - Example: `Knight/PIXELLAB_KNIGHT.md`
  - For any new character, create `src/assets/<CharacterName>/` and put their PixelLab/design doc there.
